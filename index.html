What is "software?"

Think of software as instructions that tell your computer's hardware, its physical components, how to work. It comprises all programs, procedures, and routines associated with the operation of its system.

What is "software development?"

According to IBM Research: "Software development refers to a set of computer science activities dedicated to the process of creating, designing, deploying and supporting software."

Software development comes into play when a team takes "raw" code, run it, test it, correct errors, and further develop it. If humans could create flawless, error-free systems, then there would be no need for software development. The fact is that software programs are written by mistake-making humans and not machines.

What do we learn in class?

In this course, we learn how to create interactive websites, either individually or collaboratively. There are three major languages we must learn: HTML, CSS, and JavaScript.

  • HTML defines the content of webpages.
  • CSS specifies the layout of webpages.
  • JavaScript programs the behavior of webpages.

JavaScript allows the user to interact with the webpage in a variety of ways, like with the input boxes and logo at the top. With just a click you can change the font, color, or backround instantly.
